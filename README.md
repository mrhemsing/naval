# Selfish Morality App

Interactive editorial prototype for the 11-scene "Selfish Morality" experience.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Current state

- Next.js + TypeScript + Tailwind scaffold
- Locked text-driven slide model
- Desktop click navigation + keyboard support
- Mobile swipe support
- Progress indicator + staged text reveal
- Ambient scene motion using the source slide artwork as reference imagery

## Notes

The wording is intentionally preserved exactly from `../content-lock.md`.
Visuals are currently based on the provided source slide images and are set up to be replaced later with rebuilt or regenerated layered art.
